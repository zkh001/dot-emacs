* 基本事項
- C-x  = Ctrl + x キーを押す
- M-x  = Alt + x キーを押す
- 基本(デフォルト)はC-xだけど，私はC-sに割り当てるよ
- 便利なバインドはC-qがプレフィックスだよ
- プログラミングとかのメジャーモードはM-pがプレフィックスだよ

* 各モードのキーバインド
** ddskk
  :PROPERTIES:
  :CUSTOM_ID: ddskk
  :END:
| key     | command                |
|---------+------------------------|
| C-q C-s | SKK (ddskk)を起動/無効 |
| C-o     | 英字入力 => かな入力   |
| q       | カナ入力on/off         |
| l       | 英字入力               |
